# Jin
The Jin Game Engine.

# Feature
+ Basic Quad Rendering
+ Basic Textures
+ Basic Transforms
+ Basic Shaders
+ Basic Input (keyboard, mouse)
+ Basic Camera
+ Basic Materials

# Build
To Build Successfully.
Needs vcpkg https://github.com/microsoft/vcpkg.
	Install glfw3, glew, glm, stb.
Created using Visual Studio 2019 Build Tools.
